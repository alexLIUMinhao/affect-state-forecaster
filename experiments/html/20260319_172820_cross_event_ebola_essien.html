这轮实验在验证什么

实验问题: 验证 held-out event=ebola-essien 时各模型的 cross-event 泛化能力。

成功标准: affect_state_forecaster 在 held-out event 上保持与主实验一致的领先趋势。

失败标准: 若 held-out event 掉点显著，则后续优先做泛化与正则化。

当前实验进展解释

  • 当前主实验里，`temporal_baseline` 是 MAE 最优模型，说明这一轮最有用的建模方向暂时偏向 `temporal_baseline`。
  • 已经验证：早期观测对未来情绪预测是有信息量的，任务本身可以继续做。
  • 当前只部分验证了 affect-state 主线：它优于最弱基线，但还没有稳定压过结构基线。
  • 已经验证：时间或结构信息至少有一部分是有效的，尤其需要关注结构特征的稳定贡献。
  • 目前还没有足够的多 ratio 证据，因此还不能判断早观察和晚观察之间的规律。
