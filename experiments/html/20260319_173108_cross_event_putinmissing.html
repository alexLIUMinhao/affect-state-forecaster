这轮实验在验证什么

实验问题: 验证 held-out event=putinmissing 时各模型的 cross-event 泛化能力。

成功标准: affect_state_forecaster 在 held-out event 上保持与主实验一致的领先趋势。

失败标准: 若 held-out event 掉点显著，则后续优先做泛化与正则化。

当前实验进展解释

  • 当前主实验里，`text_baseline` 是 MAE 最优模型，说明这一轮最有用的建模方向暂时偏向 `text_baseline`。
  • 已经验证：早期观测对未来情绪预测是有信息量的，任务本身可以继续做。
  • 当前没有验证 affect-state 主线，需要先做诊断，再决定是否保留它作为论文核心方法。
  • 目前还没有足够的多 ratio 证据，因此还不能判断早观察和晚观察之间的规律。
  • 这轮实验存在需要优先解释的异常现象，因此后续应先做诊断实验，而不是直接扩大模型复杂度。
